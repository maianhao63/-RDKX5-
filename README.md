# -RDKX5-
项目简介
本作品是一款专为智能仓储场景研发的自动化作业小车，采用 RDK X5 作为核心控制板，构建 ROS2 上位机与 ESP32 下位机协同架构，通过 WiFi 实现上下位机无线通讯，搭配四驱底盘，打造出高效稳定的仓储作业设备。
小车以 ROS2 为核心控制系统，实现实时目标识别、自主导航与智能避障，能精准感知仓库环境，灵活规划行驶路径，规避各类障碍物，保障四驱底盘平稳安全运行。同时搭载六自由度机械臂，可灵活完成货物抓取、转运、码放等精细化操作，满足仓储货物搬运、整理的多元需求。
硬件架构
上位控制单元：RDK X5 开发板
AI 视觉目标识别、图像采集处理
ROS2 导航、路径规划、避障算法运行
六轴机械臂运动解算、任务逻辑调度
下位驱动单元：ESP32 单片机
四驱底盘电机驱动、速度闭环控制
WiFi 无线数据透传，接收上位机运动指令
底层传感器信号采集与反馈
执行机构
四轮独立驱动运动底盘
六自由度机械臂（货物抓取、转运、码垛）
通信方案：WiFi 无线通信，完成 ROS2 上位机与 ESP32 下位机指令交互、状态回传
核心功能
✅ 实时视觉目标识别：识别仓储货物、货架、动态障碍物
✅ 自主 SLAM 建图导航：仓库地图构建、定点自主行驶
✅ 智能动态避障：障碍物实时检测，自动重规划行驶路径
✅ 四驱底盘精准运动控制：差速转向、平稳启停调速
✅ 六轴机械臂精细化作业：货物抓取、转运、分层码放
✅ 上下位机无线协同调度：ROS2 指令下发、底层设备状态回传
项目目录结构


环境依赖
上位机 RDK X5
ROS2 Humble
RDK X5 开发套件 SDK
Maixcampro、色块识别、串口传输
SLAM 导航算法功能包
下位机 ESP32
Arduino / ESP-IDF 开发环境
WiFi TCP/UDP 通信基础库 micros功能包
直流电机驱动控制库
快速启动步骤
硬件接线：完成 RDK X5、ESP32、底盘电机、机械臂、相机全部硬件接线
下位机烧录：将 esp32 目录程序烧录至单片机，修改代码内 WiFi 账号密码
上位机编译：在 RDK X5 设备内编译 src 下全部 ROS2 功能包
通信节点启动：建立 ROS2 与 ESP32 无线数据连接
导航视觉运行：加载仓库地图，启动目标识别与自主行驶功能
机械臂任务调度：下发抓取、转运、码放自动化作业指令
适用场景
智能仓库货物全自动转运
仓储货架物料分拣、自动码垛
仓库环境巡检、物料盘点辅助作业
